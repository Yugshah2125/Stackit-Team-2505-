How to join 2 columns in a data set to make a separate column in SQL

sql join beginner
I do not know the code for it as I am a beginner. As an example what I need to do is like there is a column 1 containing First name and column 2 consists of last name. I want a column to combine
Answers
1
Answer 1
You can use SQL functions to combine two columns:
  • SELECT FirstName || ' ' || LastName AS FullName FROM your_table; (for PostgreSQL, SQLite)
  • SELECT CONCAT(FirstName, ' ', LastName) AS FullName FROM your_table; (for MySQL, SQL Server)
